## OCR алгоритм

1) Создание OnnxRunTime сессий
2) Детекция текстовых областей:
   - Предобработка изображения: изменение размера (960 х 960), нормализация RGB каналов, транспонирование тензора
   - Инференс det.onnx -> получение карты вероятностей (prob_map)
   - Постобработка (postprocess_db.py): prob_map -> binary -> contours -> score -> minAreaRect -> boxes
3) Фильтрация текстовых областей
4) Распознавание текста в каждом текстовом блоке:
   - Предобработка: вырезание текстовых областей, коррекция перспективы, масштабирование, транспонирование
   - Инференс rec.onnx -> матрица вероятностей для каждого индекса текстовой строки по символам из ключей charset.txt
   - Декодирование: argmax на каждом срезе в матрице вероятностей
